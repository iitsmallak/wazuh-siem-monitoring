# Wazuh SIEM Monitoring Project

## 📌 Overview
This project demonstrates a Security Information and Event Management (SIEM) setup using Wazuh to monitor, detect, and analyze security events in a Linux environment.

##  Objectives
- Implement centralized security monitoring
- Detect suspicious activities and threats in real-time
- Analyze system and network logs
- Improve incident detection and response skills

##  Technologies Used
- Wazuh SIEM
- Linux (Ubuntu / CentOS)
- ELK Stack (Elasticsearch, Logstash, Kibana)
- Virtualization (VMware / VirtualBox)

## Architecture
- Wazuh Manager
- Wazuh Agent(s)
- Dashboard for log analysis

## Features
- Real-time log monitoring
- Security alerts and rule-based detection
- File integrity monitoring
- Vulnerability detection (if enabled)

## What I Learned
- SIEM architecture and components
- Log analysis and correlation
- Threat detection techniques
- Security monitoring in Linux systems
